ADAGRAMS SWAP MEET TASK LIST API

An anagram is a word or phrase formed by rearranging the letters of a different word or phrase. In this project you and your partner will be creating Adagrams, a game in which a player is given a random set of letters and must make an anagram using those letters. Each submitted word will score points.

A swap meet is an event that requires buyers and merchants to make an equal exchage. In this project the main goal was to have multiple categories that consisted of items with different conditiond and swapping them with the vendors based on it's condition or category.

The task list project was an introduction with API's and SQLalchemy as a database. The main objective of the project was to pratice CRUD operations and updating or deleting information from a task-list once it was complete or updated.It also allowed us to see HTTP requests live in Postman and how to correct errors.
